# 🌍 GeoSphere India
An interactive Earth Science learning platform for Indian geology, geomorphology, minerals, tectonics and geography.

## Features
- Interactive Geological Map of India
- Plate Tectonics Simulator
- Mineral & Rock Explorer
- Live Earthquake Dashboard (USGS API)
- Geological Time Scale
- Soil & Economic Geology
- BHU Semester Notes
- Geo Quiz & Flashcards

## Tech Stack
Python · Streamlit · Plotly · Pandas · USGS API

## Run Locally
pip install streamlit plotly pandas requests
streamlit run main.py
